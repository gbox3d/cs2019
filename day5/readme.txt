조건문
=> 조건식에 대한 연산자 => >,<,=, &&, ||, ! 
=> num1 연산자 num2 = 계산결과 => 1+2=3
=> 1 > 0 = true(1) , 1 < 0 = false(0)
=> 관계 연산자 => <,>,=,>=,<=,!=,==
=> 논리 연산자 => &&(논리곱),||(논리합), !(부정)
