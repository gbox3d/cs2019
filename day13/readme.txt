519p. 파일 입출력

파일 => 외부디바이스의 추상적 개념(소프트웨어적)
=> 쓰기 => pc->디바이스(하드디스크,플로피디스크) 로 전송
=> 읽기 => 디바이스-> pc 전송
=> I/O => event 데이터를 받았을때 통지=> 접속,종료 시점 구분

파일 열기 
=> FILE * => 파일 포인터 객체
=> FILE *fopen (파일이름,오픈형식(r,w,a) )
